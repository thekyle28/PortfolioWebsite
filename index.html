Elucidate

October 2015

For BrumHack 3.0 myself, Tom Clarke, Sebastian Dica, and Cameron Angus created an application that utilised Clarifai's API, which returned words associated with a picture and probabilities for how accurate each one was. For example if a picture of a dog was sent to the API then Clarifai would return words such as, dog, animal, fur, eyes, tongue etc. with dog and animal having the highest probabilities. We utilised the words returned from a picture that the user submitted and searched on Wikipedia for articles related to those words. We would then display an interactive visualisation of all of the Wikipedia articles received from the picture using D3. An image of the application in action can be seen, this was the result of searching with a picture of the clock tower in the University of Birmingham, Old Joe. Elucidate cannot be used currently because the API for Clarifai is limited to use within the Hackathon.
My role during the Hackathon was to link the Clarifai API to Elucidate to allow us to return words for photos. This proved to actually be a lot more difficult than one would have thought and it involved creating a Java Maven project from scratch and using a local host on my PC. With assistance from the Clarifai representative we managed to eventually get the API working. Finally as we brought each of the tasks that we were working on together, we created a website for the application and I assisted with this by pair programming to work on the HTML, CSS and JavaScript of the website. This was one of my stand-out Hackathons for being so productive, whilst also having a lot of fun.

Curse of Life

February 2016

For Global Games Jam (GGJ) 2016, myself, Hung Hoang, Vince Hoang and Barbra Ferreira created a story driven puzzle game. GGJ is a 48 hour event where you create games and the theme for GGJ this year was ritual. The main premise of the game was that you played as Tristan who has been banished to limbo and cursed with immortality. Tristan wants to die, and he has tried many different ways, however he can only kill himself by performing a ritual. In order to perform this ritual Tristan must kill 3 demons. As you play through the game you find out that these demons are in fact related to Tristan some way; you'll have to play the game to find out why. In order to complete Curse of Life we split the team evenly, with Vince and Barbra focusing on the Art, Sound and Design of the game, whilst Hung and I focused on the programming. I programmed the minimap, the dialogue boxes including all of the dialogue for the game and the heart beat and death logic for Tristan. I learnt a lot from this event which included a lot of coding in Unity, and it was the first time I had developed a game with a team of people, which proved to be very effective.
